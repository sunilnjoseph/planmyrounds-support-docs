# PlanMyRounds Docs

Public documentation for PlanMyRounds — rostering software for clinical teams, hospitalist groups, and on-call rotations.

Live site: https://planmyrounds.mintlify.app

## Structure

- `docs.json` — site config (branding, navigation, theme)
- `index.mdx` — landing page
- `quickstart.mdx` — getting started
- `shifts/` — Shifts module pages
- `images/` — logos and assets

## Editing

Edit via the web editor at https://dashboard.mintlify.com. Changes commit to this repo and deploy automatically.

For local preview:

```bash
npm i -g mint
mint dev
